# Proxmox server: requirements

[Link](https://www.proxmox.com/en/proxmox-ve)

## Required for getting started

- A server
- Virtualization support for CPU
- 64-but CPU
- fast storage (as fast as possible)
- NIC

## RAM and CPU

Official minimum: 

| RAM (GB) | CPU (Cores) |
| --- | --- |
| 1 | - | 

Recommended minimum:

| RAM (GB) | CPU (Cores) |
| --- | --- |
| 4 | - |
